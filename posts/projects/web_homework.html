📝 Assignment Index - Lập trình Web

Tổng hợp các bài tập lập trình web với HTML, CSS, JavaScript

Web Development HTML/CSS JavaScript Projects

Chào các bạn! Mình là Vũ Văn Thông, sinh viên năm 3 ngành An toàn thông tin. Đây là tổng hợp các bài tập lập trình web mà mình đã hoàn thành trong quá trình học tập. Các bài tập bao gồm HTML, CSS, JavaScript và các kỹ thuật web development cơ bản đến nâng cao.

1) Giới thiệu về các Assignments

Trong môn học Lập trình Web, mình đã hoàn thành nhiều bài tập khác nhau, từ cơ bản đến nâng cao. Mỗi assignment giúp mình rèn luyện các kỹ năng quan trọng trong web development.

Các kỹ năng đã học:

  • HTML5 - Semantic markup, forms, multimedia
  • CSS3 - Flexbox, Grid, animations, responsive design
  • JavaScript - DOM manipulation, events, ES6+
  • Responsive Design - Mobile-first approach
  • Web APIs - Fetch, LocalStorage, Geolocation

2) Danh sách Assignments

Assignment 1: HTML Basics

Bài tập đầu tiên về HTML cơ bản, tạo cấu trúc trang web với các thẻ semantic.

  • Sử dụng các thẻ HTML5 semantic
  • Tạo forms với validation
  • Nhúng multimedia (images, videos)
  • Tạo tables và lists

Assignment 2: CSS Styling

Học cách styling trang web với CSS, sử dụng selectors, properties và layouts.

  • CSS Selectors và Specificity
  • Box Model và Positioning
  • Flexbox và Grid Layout
  • CSS Animations và Transitions

Assignment 3: Responsive Design

Tạo trang web responsive với media queries và mobile-first approach.

  • Media Queries
  • Flexible Images và Videos
  • Mobile Navigation
  • Responsive Typography

Assignment 4: JavaScript Basics

Học JavaScript cơ bản: variables, functions, control flow.

  • Variables và Data Types
  • Functions và Scope
  • Conditional Statements
  • Loops và Arrays

Assignment 5: DOM Manipulation

Thao tác với DOM để tạo trang web interactive.

  • Selecting Elements
  • Modifying Content và Attributes
  • Event Listeners
  • Creating và Removing Elements

Assignment 6: Forms và Validation

Tạo forms với validation sử dụng JavaScript.

  • Form Elements
  • Client-side Validation
  • Regular Expressions
  • Error Handling

Assignment 7: AJAX và APIs

Làm việc với APIs sử dụng Fetch API.

  • Fetch API
  • Promises và Async/Await
  • JSON Parsing
  • Error Handling

Assignment 8: Final Project

Dự án cuối kỳ tổng hợp tất cả kiến thức đã học.

  • Full-stack Web Application
  • Responsive Design
  • Interactive Features
  • API Integration

Qua quá trình làm các assignments, mình đã học được nhiều điều:

  • Semantic HTML giúp code dễ đọc và SEO tốt hơn
  • CSS Flexbox và Grid làm layout dễ dàng hơn nhiều
  • Mobile-first approach giúp trang web responsive tốt hơn
  • JavaScript làm trang web trở nên interactive và dynamic
  • APIs mở ra nhiều khả năng tích hợp dữ liệu
